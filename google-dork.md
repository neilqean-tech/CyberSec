## 🧠 What is Google Dorking?

Google Dorking is a technique that uses advanced search operators to find specific information on the internet.

It can be used to:

* Find exposed files
* Discover login pages
* Identify vulnerable websites

---

## ⚙️ Basic Google Dorks

### 1. Find specific file types

```bash
filetype:pdf cybersecurity
```

* Finds PDF files related to cybersecurity

---

### 2. Find pages with keywords in URL

```bash
inurl:login
```

* Finds login pages

---

### 3. Find pages with specific titles

```bash
intitle:"index of"
```

* Finds open directories

---

### 4. Search within a specific website

```bash
site:example.com
```

* Searches only inside that website

---

### 5. Find exposed files

```bash
site:example.com filetype:txt
```

* Looks for text files on a website
